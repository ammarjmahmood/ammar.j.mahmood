EcoSnap: AI-Powered Recycling Solution

The purpose of this personal assistant application is to empower users to make environmentally conscious decisions. Using technologies like Ray-Ban Meta smart glasses, OpenAI’s APIs, and machine learning, EcoSnap helps users reduce their carbon footprint by offering eco-friendly alternatives to what they eat and buy.

Inspiration: We were inspired by the idea of integrating sustainability into daily decisions, offering real-time insights to simplify eco-friendly choices. Our goal is to make sustainability a natural part of daily life by leveraging AI and smart devices.

What It Does:

  • Eco-Friendly Shopping Companion: Uses barcode/QR code scanning for sustainability ratings and eco-friendly alternatives.
  • Eco-Friendly Food Companion: Analyzes food items for their environmental impact and offers sustainable alternatives based on user preferences.
  • Eco-Friendly Waste Reduction Guide: Provides location-based recycling guidance, waste reduction tips, and information on local recycling centers.
  • Sustainability Ratings & Leaderboard: Users receive a daily sustainability score and can participate in eco-challenges with leaderboards.

How We Built It:

We built EcoSnap using technologies like OpenAI API, Node.js, JavaScript, HTML, CSS, machine learning, and Python.

Challenges:

One of the biggest challenges was working with Meta Smart glasses, as there was no SDK. We created workarounds by setting up a web server and implementing APIs.

What We Learned:

We learned a lot about API integration, front-end, and back-end development. In hindsight, more thorough research and tool familiarity could have made the development smoother.

What's Next for EcoSnap:

  • Recycling Center Information (Future)
  • Waste Reduction Tips (Future)
  • Location-Based Recycling Guidance (Future)
